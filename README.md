# NFT

## Setup
 - npm i / yarn
#### In seperate terminals:
 - ipfs daemon
 - ganache-cli -d
 - truffle console

### Inside truffle console
 - compile
 - migrate
 - nft = await NFT.deployed()

### Finally
 - connect Metamask wallet to localhost:8545
 - npm/yarn start
